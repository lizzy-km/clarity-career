# **App Name**: ClarityCareer

## Core Features:

- Job Listings: Display job listings with filters for location, industry, and salary range.
- Company Reviews: Allow users to submit and view company reviews, ratings, and culture insights. This will be stored in Firestore.
- Salary Data: Provide salary data aggregated from user submissions, searchable by job title and location. This will be stored in Firestore.
- Interview Information: Enable users to share and read interview experiences, questions, and difficulty levels. This will be stored in Firestore.
- User Authentication: Implement secure user authentication with Firebase Authentication, supporting both email/password and OAuth.
- Job Application Tracking: Allow users to save job postings and track their application status.
- Profile Management: Enable users to manage their profiles, including work experience, education, and skills.

## Style Guidelines:

- Primary color: Deep blue (#224578) to convey trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Teal (#26A69A) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look, is used for both headlines and body text.
- Use simple, professional icons for navigation and key features.
- Design a clean, intuitive layout with a focus on readability and easy navigation.
- Incorporate subtle animations for a polished user experience.
